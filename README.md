# Candida Diet Planner

A comprehensive app to help users follow the 3-phase Candida Diet and Cleanse protocol. This application provides detailed information about each phase of the diet, meal planning tools, recipe suggestions, and progress tracking functionality.

## Features

- **Detailed Phase Information**: Learn about each phase of the Candida Diet, including the 14-Day Cleanup, MEVY Diet, Low Allergy Diet, and Food Reintroduction.
- **Meal Planner**: Plan your meals for each day of the week based on your current phase.
- **Recipe Database**: Browse and search through a collection of candida-friendly recipes filtered by phase and meal type.
- **Progress Tracker**: Monitor your symptoms, track improvements, and record food reintroductions to identify sensitivities.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/candida-diet-app.git
cd candida-diet-app
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```

4. Open your browser to http://localhost:3000 to view the app.

## Usage

### Home Page
The home page provides an overview of the Candida Diet and links to each phase of the cleanse.

### Phase Information
Each phase page contains detailed information about the dietary guidelines, allowed and restricted foods, sample meal plans, and tips for success.

### Meal Planner
1. Select your current phase
2. Navigate through weeks using the arrows
3. Add meals for each day by clicking the "Add" buttons
4. Review meal suggestions specific to your phase

### Recipe Database
1. Search for recipes using the search bar
2. Filter recipes by phase and meal category
3. View detailed recipe information including ingredients and instructions

### Progress Tracker
1. Track your symptoms by adjusting the sliders based on severity
2. Add new symptoms to track
3. Record journal notes about your progress
4. In Phase 3, log food reintroductions and any reactions
5. Monitor your overall improvement with the progress chart
